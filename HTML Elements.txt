Basic HTML Elements

Example:
<html>
<body>

<h1>My first heading. </h1>
<p>My first paragraph.</p>

</body>
</html>

There are 4 elements in total, called a nested HTML elements.
Each element should looks like <start> content </start>
*Notice: We have element that can be empty. For these elements, 
we don't need the end tag. <br>

Example: 
<p>This is a <br> paragraph with a line break. </p>
